QR Code Officiel

QR code authentifié à apposer sur vos tickets

Authentifié HMAC-SHA256

Ce QR code est unique et généré depuis votre clé secrète.
Tout scan depuis cette app retournera Authentique.

Comment l'utiliser

  1. 1 Téléchargez le QR code et imprimez-le sur vos tickets.
  2. 2 Lors d'un contrôle, scannez le ticket avec la page Scanner.
  3. 3 L'app affiche ✓ Authentique ou ✗ Faux.
